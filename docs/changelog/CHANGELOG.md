## [0.0.1] - 2026-04-07 - ENRIQUE GOMEZ

### Added
- Estructura inicial del proyecto bajo el esquema `src-layout`.
- Directorios para el código fuente: `eeea_py/`, incluyendo subpaquetes `algorithms/`, `core/` y `utils/`.
- Directorio de pruebas unitarias `test/` con inicialización para `pytest`.
- Directorios de soporte `docs/` para documentación técnica y `examples/` para guías de uso.
- Archivos de configuración base:
    - `pyproject.toml` para la gestión de dependencias y empaquetado.
    - `.gitignore` configurado para entornos de Python y VS Code.
    - `README.md` con la descripción general del proyecto.
    - `CONTRIBUTING.md` con las guías de estilo PEP 8 y PEP 257.
    - `requirements-dev.txt` y `requirements.txt` para herramientas de desarrollo.
- Implementación de archivos base en el paquete principal: `__init__.py`, `__main__.py` y `config.py`.
